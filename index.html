Java Install

Some PCs might have Java already installed , Java sure is in your PC if you play Minecraft

To check if you have Java installed on a windows PC, search Java in start bar or type in following in command prompt

C:\Users/Your Name>java-version>

If Java is installed, you will see something like this (depending on the version):

java version "11.0.3" 2018-10-166 LTS Java(TM) SE Runtime Environment 18.9 (build 11.0.1+13-LTS) Java HotSpot(TM) 64-Bit Server VM 18.9 (build 11.0.1+13-LTS, mixed mode)

Java-Syntax
public class Main { public static void main(String[] args) { System.out.println("Hello World"); } }

Every line of code that runs in Java must be inside a class. In our example, we named the class Main. A class should always start with an uppercase first letter. Note: Java is case-sensitive: "MyClass" and "myclass" has different meaning.

The name of the java file must match the class name. When saving the file, save it using the class name and add ".java" to the end of the filename. Themain() method is required and you will see it in every Java program: public static void main(String[] args) Any code inside the main() method will be executed. You don't have to understand the keywords before and after main just yet.Nevertheless they are simple public is called access modifier and is used to set scope of the class like a method with access modifier of protected can only be accessed within the same package , static means the method is static , void means the return type is null and main is the name of the method For now, just remember that every Java program has a class name which must match the filename, and that every program must contain the main() method.
